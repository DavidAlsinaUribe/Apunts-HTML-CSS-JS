Estructura documento HTML5

la estructura básica es la siguiente:

            <!DOCTYPE html>
            <html lang="en">
                    <head>
                            <meta charset="UTF-8">
                            <title>Document</title>
                            <style> Aquí va el estilo </style>
                    </head>
                    <body>
                    </body>
            </html>
La etiqueta 'head contiene toda la configuración del archivo: título e icono en la pestaña de navegador, conjunto de carácteres, etc. La etiqueta 'body' contiene el cuerpo del documento. Los comentarios en un documento HTML se escriben dentro de <!--comentario-->. Deentro de una etiqueta o en un documento CSCSS,se escriben dentro de /* comentario */.
